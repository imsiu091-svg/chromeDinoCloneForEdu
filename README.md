# chromeDinoCloneForEdu
크롬 공룡게임을 웹으로 클론코딩하기.

> 모든 캐릭터는 도트로 제작한다. 순수 HTML, CSS로 제작

# 캐릭터
* 주인공: 공룡 캐릭터

# 조작 기능 - 컨트롤
* 달리기(기본 기능)
  * 시간이 지나면 속도가 빨라짐
* 점프 기능
  * 키보드 입력: `Space` `위쪽 방향키` 
* 웅크리기 기능
  * 키보드 입력: `아래 방향키`

# 맵
사막 컨셉. 시간에 따라 낮과 밤이 바뀜.(점수 700점 마다)

# 오브젝트
* 선인장
  * 주인공이 점프해서 피해야 함
  * 랜덤 위치에서 선인장 발생
  * 작은 선인장과 큰 선인장이 있다. 초반에는 작고, 후반에는 점점 커짐
  * 일정 점수 넘으면 선인장 늘어남
* 케찰코아툴루스(익룡)을 피해아한다(200~500점 이후로 등장)
  * 주인공이 웅크려서 피해야 함

# 점수
0.05초마다 점수가 1씩 오름.
